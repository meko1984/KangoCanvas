# KangoCanvas

看護学生向けの、自由配置型看護関連図Webアプリです。

Web版: https://meko1984.github.io/KangoCanvas/

## 現在の構成

- ブラウザ内だけに関連図を自動保存
- GitHub Pagesなどの静的ホスティングに対応
- 将来のレンタルサーバー移行を妨げない構成
- アカウント・クラウド同期・サーバー送信は未実装

## 開発

```sh
pnpm install
pnpm dev
```

本番用ファイルは `pnpm build` で `dist` に作成されます。

## 個人情報

氏名、患者ID、生年月日など、個人を特定できる情報は入力しないでください。
関連図データは利用者のブラウザ内に保存されますが、ブラウザデータの削除や
端末故障によって失われる可能性があります。定期的にバックアップファイルを
書き出してください。
